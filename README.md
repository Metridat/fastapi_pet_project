# FastAPI E-commerce API
Это demo-backend интернет-магазина на Python + FastAPI, реализованный как:

# О проекте

- CRUD для категорий, продуктов, отзывов, корзины и заказов  
- Аутентификация/авторизация + роли (покупатель / продавец / админ)  
- Полноценная бизнес-логика (валидация, права доступа, управление stock, расчёт рейтингов и т.д.)  
- Ведение логов через loguru + глобальное middleware для логирования запросов — демонстрация знаний о слоях и мониторинге  
- Чёткое разделение на слои: роутеры → сервисы → репозитории → модели/схемы — чистая, поддерживаемая архитектура  
- Подключение к Postgres (с asyncpg + SQLAlchemy), поддержка миграций через Alembic  
- Контейнеризация через Docker + docker-compose 



# Стек технологий

| Слой / задача | Технологии / подход |
|--------------|---------------------|
| Web-framework / API | FastAPI |
| База данных | PostgreSQL + SQLAlchemy + asyncpg |
| Миграции | Alembic |
| Логирование | loguru + middleware |
| Авторизация / Аутентификация | JWT (access / refresh), Roles (buyer / seller / admin) |
| Контейнеризация / Dev-ops | Docker, docker-compose |
| Валидация / Схемы | Pydantic |
| REST API / CRUD + бизнес-логика | Сервисы → репозитории 
| Работа с запросами | Async / await, асинхронные сессии БД |

# Быстрый старт

1. Клонируйте репозиторий  
   ```bash
   git clone https://github.com/YourUsername/fastapi_pet_project.git
   cd fastapi_pet_project

2. Скопируйте пример окружения:
   cp .env.example .env

3. Заполните обязательные поля в .env:
   - POSTGRES_PASSWORD
   - ADMIN_PASSWORD
   - SECRET_KEY - сгенерировать
   - APP_PORT - выставить другой, если занят
   - POSTGRES_PORT - выставить другой, если занят

4. Проект использует Alembic для миграций. Запустите:
   docker compose up --build
   --Автоматически выполняются:--
      1.Создание всех таблиц
      2.Создание администратора из .env

После запуска:
- API доступно на: http://localhost:8000
- Админ создан автоматически из значений .env
